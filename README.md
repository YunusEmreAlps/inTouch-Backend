## Used Technologies
- MongoDB
- Node.js
- Express.js

## How To Install?
- 1-) Download this repository to your machine.
```bash
git clone https://github.com/YunusEmreAlps/inTouch-Backend.git
```
- 2-) Open Project Folder and download packages
```bash
npm install
```
- 3-) Running the server
```bash
node server 
```
